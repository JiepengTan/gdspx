# godot for spx

## prepare env
1. install "**make**" for build
2. install python
3. run script 
    ```
    ./scripts/init.sh
    ```


## run demo 
```
./scripts/run.sh -i tutorial/01_aircraft
```

Use the "WSAD" keys to move the player
