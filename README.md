# godot for spx

## prepare env
- download and install godot 4.2.2.statble 
- make cmd "godot" work
    - windows add godot 4.2.2.stable to system path
    - linux|mac copy to 
       ```
        sudo mv path/to/your/Godot_v4.x-stable_linux.x86_64 /usr/local/bin/godot
       ```

## run demo
### windows
```
cd cmd/template
make
make run
```

### linux | macos
```
cd cmd/template
make && make run
```
